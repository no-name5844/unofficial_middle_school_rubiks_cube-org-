# xssz 魔方非官方组织

> 一个初中魔方社团的临时章程，105条，18章，52处权力约束机制。

---

## 这是什么

一份由初中生撰写的社团章程——但读起来像一部微型宪法。

组织当前为非官方状态。一旦获得学校认可并签署合同（即"官方化"），名称中的"非官方"三字即可摘除，转为正式学生组织。若学校未来因分裂或招生不足而关闭，组织自动转为本县魔方协会，向民政局申请社会团体登记。

## 架构

```
全体成员会议（最高权力）
  ├── 常任委员会（日常执行）
  ├── 组织代表（临危决断）
  ├── 监察委员会（独立合规审查）
  ├── 经费管理委员会（独立管账，直接对全会负责）
  ├── 历史管理委员会（记录传承）
  ├── 安全管理委员会（活动安全）
  ├── 运营小组（线上网站与平台维护）
  └── 联络员制度（为无法上网者传递信息与代为投票）
```

## 核心特征

- **权力分散：** 管账的不花钱（经费委），花钱的不记账（常任委），查账的不管钱（监察委）
- **自我纠错：** 52处约束分布在105条中——从24小时会议通知到被否提案14日冷却，从利益回避到代号匿名投票
- **分阶段演进：** 初创期创始人行使临时决策权；满15人召开第一次全体成员会议；会后14日内完成所有机构的选举
- **信息防垄断：** 联络员制为不能上网的成员传话收票；档案可公开部分主动公开

## 不是AI生成的

这是一份由真实初中生一条一条讨论出来的作品。每一处权力约束都对应着上一个组织的失败教训。AI只会写出"社长说了算"——不会在魔方社团里设三权分立。
